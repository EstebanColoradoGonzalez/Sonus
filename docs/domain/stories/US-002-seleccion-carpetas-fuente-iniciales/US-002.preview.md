# Preview de Interfaz — HU #US-002: Selección guiada de Carpetas Fuente iniciales

> ⚠️ **PROPUESTA PENDIENTE DE VALIDACIÓN CON DISEÑO** — Este prototipo debe ser revisado y aprobado por el equipo de diseño antes de implementarse.
>
> Formato: **Mermaid (flujo de navegación)** · Plataforma inferida: **mobile** · Línea gráfica: *defaults* (sin proyecto frontend en el workspace).

```mermaid
flowchart TD
    %% Generado para HU #US-002: Selección guiada de Carpetas Fuente iniciales

    Prev(["Fin de US-001<br/>(permisos concedidos u omitidos)"]) --> Screen["Pantalla: Selección de Carpetas Fuente<br/>· Título + explicación (Apalancamiento 2)<br/>· Lista de carpetas seleccionadas<br/>· [Agregar carpeta]<br/>· [Continuar] (primario)"]

    Screen --> ChkList{"¿La lista tiene<br/>al menos 1 carpeta?"}

    %% Escenario 3 — obligatoriedad
    ChkList -->|"No (lista vacía)"| Empty["Empty state orientador<br/>+ [Continuar] DESHABILITADO"]
    Empty --> Screen
    ChkList -->|"Sí (>= 1 carpeta)"| Enabled["[Continuar] HABILITADO"]
    Enabled --> Screen

    %% Escenarios 1 y 2 — agregar (una o múltiples)
    Screen -->|"toca 'Agregar carpeta'"| SafPicker(["Selector SAF del SO<br/>ACTION_OPEN_DOCUMENT_TREE"])

    SafPicker -->|"elige un directorio"| ChkGrant{"¿El SO otorga el permiso<br/>persistible del treeUri?"}
    SafPicker -->|"cancela sin elegir"| CancelMsg["Aviso no intrusivo:<br/>'No se agregó ninguna carpeta'"]
    CancelMsg --> Screen

    %% Escenario 5 — permiso denegado
    ChkGrant -->|"No (permiso denegado)"| DenyMsg["Aviso: 'Se necesita acceso a la<br/>carpeta para descubrir tu audio'<br/>(ERR_PERMISSION_DENIED) · sin bucle"]
    DenyMsg --> Screen

    %% Escenario 4 — duplicado
    ChkGrant -->|"Sí"| ChkDup{"¿El treeUri ya está<br/>registrado? (unicidad)"}
    ChkDup -->|"Sí (duplicado)"| DupMsg["Aviso: 'La carpeta ya estaba agregada'<br/>(ERR_DUPLICATE_SOURCE_FOLDER · WARNING)<br/>· no crea duplicado"]
    DupMsg --> Screen

    %% Escenarios 1/2 — registro efectivo
    ChkDup -->|"No"| Register["takePersistableUriPermission<br/>+ registrar SourceFolder<br/>(treeUri, displayPath, dateAddedMs)"]
    Register --> AddedToList["Carpeta añadida a la lista<br/>(se pueden agregar más — Escenario 2)"]
    AddedToList --> Screen

    %% Escenario 6 — quitar de la lista (pre-escaneo, sin cascada)
    Screen -->|"toca 'Quitar' en un ítem"| RemoveItem["Eliminar SourceFolder<br/>+ releasePersistableUriPermission<br/>(remoción ligera, sin cascada)"]
    RemoveItem --> Screen

    %% Escenario 7 — continuar al escaneo
    Screen -->|"toca 'Continuar' (>= 1 carpeta)"| Confirm["Confirmar permisos persistidos vigentes"]
    Confirm --> Next(["Escaneo fundacional (US-003)<br/>— fuera de esta HU"])
```

## Leyenda de trazabilidad (AC → flujo)

| AC | Rama del diagrama |
|----|-------------------|
| Escenario 1 (Flujo Principal) | Agregar carpeta → SAF → otorga permiso → no duplicado → registrar → lista |
| Escenario 2 (Múltiples carpetas) | Registrar → "se pueden agregar más" → volver a la pantalla |
| Escenario 3 (Obligatoriedad ≥1) | ¿Lista tiene ≥1? → No → [Continuar] deshabilitado / Sí → habilitado |
| Escenario 4 (Duplicado) | ¿treeUri ya registrado? = Sí → aviso `ERR_DUPLICATE_SOURCE_FOLDER` |
| Escenario 5 (Cancelación / permiso denegado) | Cancela → aviso; ¿otorga permiso? = No → `ERR_PERMISSION_DENIED` |
| Escenario 6 (Quitar de la lista) | Quitar ítem → eliminar `SourceFolder` + liberar permiso (sin cascada) |
| Escenario 7 (Confirmar y transitar) | [Continuar] → confirmar permisos → escaneo (US-003) |
| Escenario 8 (Autarquía) | Invariante transversal (solo SAF; sin `MediaStore`/media runtime/red — verificable) |
