# Preview de Interfaz — HU #US-006: Remover una Carpeta Fuente

> ⚠️ **PROPUESTA PENDIENTE DE VALIDACIÓN CON DISEÑO** — Este prototipo debe ser revisado y aprobado por el equipo de diseño antes de implementarse.
>
> Formato: **Mermaid (flujo de navegación)** · Plataforma inferida: **mobile** · Línea gráfica: *defaults* (sin proyecto frontend en el workspace).

```mermaid
flowchart TD
    %% Generado para HU #US-006: Remover una Carpeta Fuente

    subgraph Entrada["Punto de entrada (operación normal · post-onboarding)"]
        Cfg(["Configuración<br/>› sección 'Carpetas Fuente'"])
    end

    Cfg --> Screen

    Screen["Pantalla: Carpetas Fuente<br/>· Lista de carpetas (displayPath + acción 'Quitar' por ítem)<br/>· [Agregar carpeta] (US-005, fuera de esta HU)"]

    %% Acción de quitar
    Screen -->|"toca 'Quitar' en una carpeta"| FetchImpact["Calcular impacto:<br/>nº tracks + nº referencias en playlists"]

    FetchImpact --> Dialog["Diálogo de confirmación<br/>· '¿Quitar carpeta X?'<br/>· 'Se eliminarán N pistas y sus referencias en M playlists'<br/>· [Confirmar] · [Cancelar]"]

    %% Escenario 8 — última carpeta
    FetchImpact -->|"es la última carpeta"| DialogLast["Diálogo de confirmación<br/>· 'Tu biblioteca quedará completamente vacía'<br/>· [Confirmar] · [Cancelar]"]

    %% Escenario 6 — cancelación
    Dialog -->|"toca 'Cancelar'"| CancelMsg["Sin cambios<br/>(ninguna eliminación)"]
    DialogLast -->|"toca 'Cancelar'"| CancelMsg
    CancelMsg --> Screen

    %% Escenario 1 — confirmación y ejecución
    Dialog -->|"toca 'Confirmar'"| Execute
    DialogLast -->|"toca 'Confirmar'"| Execute

    Execute["releasePersistableUriPermission(treeUri)<br/>+ DELETE SourceFolder<br/>→ CASCADE: Track → PlaylistTrackCrossRef,<br/>  QueueItem, PlaybackProgress<br/>→ Purga dimensiones huérfanas (Artist/Album/Genre)"]

    %% Escenario 2 — catálogo se contrae
    Execute --> Catalog["Catálogo se contrae:<br/>N tracks eliminados<br/>Tracks de otras carpetas intactos — Escenario 2"]

    %% Escenario 3 — playlists afectadas
    Catalog --> Playlists["Referencias en playlists eliminadas<br/>Playlists vacías conservadas — Escenario 3"]

    %% Escenario 4 — dimensiones huérfanas
    Playlists --> Dims["Dimensiones huérfanas purgadas<br/>(Artist/Album/Genre sin referencias)<br/>Centinelas id=1 preservados — Escenario 4"]

    %% Escenario 5 — permiso SAF liberado
    Dims --> SaftPerm["Permiso SAF liberado<br/>releasePersistableUriPermission — Escenario 5"]

    SaftPerm --> Success["Carpeta eliminada de la lista<br/>Snackbar: 'Carpeta quitada correctamente'"]
    Success --> Screen

    %% Escenario 7 — cola afectada (nota)
    Execute -.->|"si había tracks de esta carpeta en reproducción"| Queue(["Cola activa ajustada automáticamente<br/>(CASCADE sobre QueueItem)<br/>Motor adapta reproducción — Escenario 7"])

    %% Escenario 9 — carpeta no encontrada
    Screen -->|"folderId ya no existe (inconsistencia)"| NotFound["ERR_ENTITY_NOT_FOUND (ERROR)<br/>Vista refrescada desde flujo observable C2"]
    NotFound --> Screen
```

## Leyenda de trazabilidad (AC → flujo)

| AC | Rama del diagrama |
|----|-------------------|
| Escenario 1 (Flujo Principal) | Quitar → calcular impacto → diálogo → Confirmar → execute → cascade → éxito |
| Escenario 2 (Catálogo se contrae) | execute → "Catálogo se contrae: N tracks eliminados" |
| Escenario 3 (Purga en playlists) | cascade → "Referencias en playlists eliminadas; playlists vacías conservadas" |
| Escenario 4 (Dimensiones huérfanas) | cascade → "Dimensiones huérfanas purgadas; centinelas id=1 preservados" |
| Escenario 5 (Permiso SAF liberado) | → `releasePersistableUriPermission` |
| Escenario 6 (Cancelación) | diálogo → Cancelar → "Sin cambios" |
| Escenario 7 (Tracks en cola) | execute -.-> "Cola activa ajustada automáticamente (CASCADE QueueItem)" |
| Escenario 8 (Última carpeta) | FetchImpact → "es la última carpeta" → diálogo especial → Confirmar |
| Escenario 9 (EntityNotFound) | → ERR_ENTITY_NOT_FOUND → vista refrescada |
| Escenario 10 (Autarquía) | Invariante transversal (solo Room/SQLite + SAF; sin red — verificable) |
