# Preview de Interfaz — HU #US-005: Agregar una Carpeta Fuente

> ⚠️ **PROPUESTA PENDIENTE DE VALIDACIÓN CON DISEÑO** — Este prototipo debe ser revisado y aprobado por el equipo de diseño antes de implementarse.
>
> Formato: **Mermaid (flujo de navegación)** · Plataforma inferida: **mobile** · Línea gráfica: *defaults* (sin proyecto frontend en el workspace).

```mermaid
flowchart TD
    %% Generado para HU #US-005: Agregar una Carpeta Fuente

    subgraph Entradas["Puntos de entrada (operación normal · post-onboarding)"]
        Cfg(["Configuración<br/>› sección 'Carpetas Fuente'"])
        Lib(["Biblioteca<br/>› acción 'Agregar carpeta'"])
    end

    Cfg --> Screen
    Lib --> Screen

    Screen["Pantalla: Carpetas Fuente<br/>· Lista de carpetas registradas (displayPath)<br/>· [Agregar carpeta]<br/>· Aviso 'contenido pendiente de escanear' (si aplica)"]

    %% Escenario 1 y 2 — abrir selector
    Screen -->|"toca 'Agregar carpeta'"| SafPicker(["Selector SAF del SO<br/>ACTION_OPEN_DOCUMENT_TREE"])

    SafPicker -->|"cancela sin elegir"| CancelMsg["Aviso no intrusivo:<br/>'No se agregó ninguna carpeta'"]
    CancelMsg --> Screen

    SafPicker -->|"elige un directorio"| ChkGrant{"¿El SO otorga el permiso<br/>persistible del treeUri?"}

    %% Escenario 5 — permiso denegado
    ChkGrant -->|"No (permiso denegado)"| DenyMsg["Aviso: 'Se necesita acceso a la carpeta<br/>para descubrir tu audio'<br/>(ERR_PERMISSION_DENIED · ERROR) · sin bucle"]
    DenyMsg --> Screen

    %% Escenario 3 — duplicado exacto
    ChkGrant -->|"Sí"| ChkDup{"¿El treeUri ya está<br/>registrado? (unicidad exacta)"}
    ChkDup -->|"Sí (duplicado)"| DupMsg["Aviso: 'La carpeta ya estaba agregada'<br/>(ERR_DUPLICATE_SOURCE_FOLDER · WARNING)<br/>· no crea duplicado"]
    DupMsg --> Screen

    %% Escenario 4 — solapamiento (anidada/contenedora)
    ChkDup -->|"No"| ChkOverlap{"¿Se solapa con otra<br/>Carpeta Fuente?<br/>(subdirectorio o contenedora)"}
    ChkOverlap -->|"Sí (solapada)"| OverlapMsg["Aviso: 'Esta carpeta se solapa con otra ya<br/>agregada; las pistas repetidas no se duplicarán'<br/>(dedup por Track.uri) · PERMITE registrar"]
    OverlapMsg --> Register

    %% Escenario 1 — registro efectivo
    ChkOverlap -->|"No (disjunta)"| Register["takePersistableUriPermission<br/>+ registrar SourceFolder<br/>(treeUri, displayPath, dateAddedMs)"]
    Register --> Preserved["Catálogo y carpetas previas<br/>SE PRESERVAN (sin reindexar) — Escenario 2"]
    Preserved --> Pending["Aviso: 'Contenido pendiente de escanear'<br/>+ acción [Iniciar re-escaneo]"]

    %% Escenario 6 — escaneo manual (frontera hacia US-007)
    Pending -->|"toca 'Iniciar re-escaneo'"| Next(["Escaneo / re-escaneo (US-007 / TRG-LIB-03)<br/>— fuera de esta HU"])
    Pending -->|"no lo inicia"| Screen

    %% Escenario 7 — persistencia
    Register -.->|"tras reinicio de la app / proceso"| Persisted(["SourceFolder + permiso SAF<br/>siguen vigentes (memoria durable)"])
    Persisted -.-> Screen
```

## Leyenda de trazabilidad (AC → flujo)

| AC | Rama del diagrama |
|----|-------------------|
| Escenario 1 (Flujo Principal) | Agregar → SAF → otorga permiso → no duplicado → no solapada → registrar → carpeta en lista |
| Escenario 2 (Preservar biblioteca) | Registrar → "Catálogo y carpetas previas se preservan (sin reindexar)" |
| Escenario 3 (Duplicado exacto) | ¿treeUri ya registrado? = Sí → aviso `ERR_DUPLICATE_SOURCE_FOLDER` |
| Escenario 4 (Solapamiento) | ¿Se solapa? = Sí → aviso y **permite** registrar (dedup por `Track.uri`) |
| Escenario 5 (Cancelación / permiso denegado) | Cancela → aviso; ¿otorga permiso? = No → `ERR_PERMISSION_DENIED` |
| Escenario 6 (Escaneo manual) | "Contenido pendiente de escanear" → [Iniciar re-escaneo] → US-007 (fuera de la HU) |
| Escenario 7 (Persistencia tras reinicio) | Registrar -.-> "SourceFolder + permiso SAF vigentes tras reinicio" |
| Escenario 8 (Autarquía) | Invariante transversal (solo SAF; sin `MediaStore`/media runtime/red — verificable) |
